1. Pendahuluan

Kami menghormati privasi Anda. Aplikasi Tasbih Digi dibuat sebagai alat bantu ibadah yang sederhana. Kebijakan Privasi ini menjelaskan bahwa kami tidak mengumpulkan data pribadi apa pun dari pengguna kami.

2. Pengumpulan Data

  • Data Pribadi: Kami tidak meminta, mengumpulkan, atau menyimpan nama, alamat email, nomor telepon, atau lokasi Anda.
  • Data Perangkat: Kami tidak mengakses kontak, galeri foto, atau sensor sensitif lainnya di perangkat Anda.
  • Data Penggunaan: Aplikasi ini bekerja sepenuhnya secara offline. Kami tidak melacak aktivitas penggunaan Anda.

3. Izin Perangkat (Permissions)

Aplikasi ini mungkin memerlukan izin berikut untuk berfungsi:

  • Vibration (Getar): Digunakan hanya untuk memberikan umpan balik (haptic feedback) saat Anda menyentuh tombol hitung.

4. Keamanan

Karena kami tidak mengumpulkan data apa pun, tidak ada data pribadi Anda yang berisiko bocor dari server kami. Semua data hitungan tasbih disimpan secara lokal di perangkat Anda sendiri.

5. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan berlaku segera setelah dipublikasikan di halaman ini.
